Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em casa caso.

Tags de Cabeçalho

<!DOCTYPE html>

Tag responsável por informar ao site que o tipo de arquivo é html5 (HyperText Markup Language) em seguida dela temos a tag <html> com o atributo lang="pt-br" (Language) que informa para o buscador qual pais é o site.

<head>

Essa tag não é vísivel para o usuario, porém nela você colocará informaçoes importantes no seu site, além disso, todas tags de configurações irão dentro desta tag (Header).

<meta>

Essa tag fornece informações para buscadores, como os tipos de caracteres que o site utiliza (UTF-8) o tamanho do site o tipo de visualização.

<title>

Essa tag somente contém o Título da página, que irá aparecer na janela do navegador

<link>

Essa tag é responsável por referênciar um arquivo externo, nela você utilizará os atributos (release) rel="tipodearquivo" e href (referência) href="Arquivo/Referênciado" os mais comuns são, css e favicon, respectivamente link:css ou link:favicon.


Tags sem Atributos

<br>

Tag responsável por quebra de linha (break row)

<hr>

Tag usada para criar uma linha horizontal (horizontal row)

<p>

Tag usada para criar um parágrafo (paragraph)


Tags com Atributos

<img>

Tag usada para criar uma imagem (image), é necessário os atributos src="Caminho/da/imagem" e alt="Descrição da imagem"
